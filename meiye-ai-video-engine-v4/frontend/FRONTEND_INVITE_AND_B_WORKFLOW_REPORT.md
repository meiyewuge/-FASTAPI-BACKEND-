## 吴哥验收操作顺序

1. 打开 `http://8.152.169.71:19181/`
2. 点击"管理员发码" → 输入 ADMIN_KEY → 生成 1 个邀请码 → 复制
3. 返回登录 → 输入手机号 + 刚复制的邀请码 → 进入系统
4. 视频库选一条已有母视频 → 点"🔁 用此裂变"
5. 选策略 + 设数量=1 → 点"🔁 B台·裂变（0元/条）"
6. 等 2-3 秒任务完成 → 播放裂变视频
7. 点下载 → 保存到本地
8. 勾选视频 → 点"📥 导出视频" → 下载到本地
